# 🚀 Machine Learning Projects — Phase 1

This repository contains my **foundational machine learning projects**, where I focused on understanding core concepts such as:
- Data preprocessing
- Feature engineering
- Model training & evaluation
- Basic ML algorithms

Each project is built using Python and Jupyter Notebook.

---

## 📂 Projects

### 🛰️ Asteroid Hazard Classifier
- **Goal:** Predict whether an asteroid is hazardous or not
- **Techniques:** Classification models (Logistic Regression, Random Forest)
- **Focus:** Binary classification & feature importance

---

### 🌍 Exoplanet Hunter
- **Goal:** Detect potential exoplanets based on astronomical data
- **Techniques:** Supervised learning models
- **Focus:** Pattern recognition in scientific datasets

---

### 🏥 Patient Health Risk Predictor
- **Goal:** Predict patient risk levels based on medical attributes
- **Techniques:** Classification algorithms
- **Focus:** Real-world healthcare prediction problem

---

### 🏠 Real Estate Price Predictor
- **Goal:** Predict property prices based on features
- **Techniques:** Regression models
- **Focus:** Understanding regression and feature relationships

---

### 🌌 Space Object Clustering
- **Goal:** Group celestial objects into clusters
- **Techniques:** K-Means / Unsupervised Learning
- **Focus:** Clustering and pattern discovery
